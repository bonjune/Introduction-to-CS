## Install Guide

To run your code in your machine, please install cs101 library in your anaconda environment.

1. [Click here to donwload cs101 library](http://cs101.kaist.ac.kr)
2. Place the python files under "/anaconda3/lib/python3.x/site-packages/"
3. Done! To check whether the files are located properly, run the command below.
```
python -c "import cs1robots"
```
